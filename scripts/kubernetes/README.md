# Kubernetes Scripts

Scripts for Kubernetes pod troubleshooting, namespace resource reporting, and read-only cluster visibility.

Scripts in this folder should document required `kubectl` context and namespace behavior.

## Available Scripts

| Folder | Purpose |
| --- | --- |
| `pod-status-summary/` | Show pods that are not Running or Completed in a namespace. |
| `namespace-resource-check/` | Show deployments, pods, services, and events in a namespace. |
