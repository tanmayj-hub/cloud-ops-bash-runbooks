# AWS Scripts

Scripts for safe AWS CLI inventory, audit, and reporting workflows.

Scripts in this folder use read-only operations unless reviewed, and examples must never include real account data.

## Available Scripts

| Folder | Purpose |
| --- | --- |
| `ec2-status-report/` | Generate a read-only EC2 instance status report. |
| `s3-public-access-check/` | Check S3 bucket public access block configuration without claiming a full ACL or bucket policy review. |
