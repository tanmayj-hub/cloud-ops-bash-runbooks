# Linux Admin Scripts

Scripts for system health, disk usage, users, groups, process review, permissions, and service troubleshooting.

Scripts in this folder are designed to be safe, documented, testable, and explainable for Linux Admin interviews.

## Available Scripts

| Folder | Purpose |
| --- | --- |
| `disk-usage-check/` | Check mounted filesystem disk usage against a threshold. |
| `top-processes-report/` | Show top CPU and memory consuming processes. |
| `user-group-audit/` | Report local users, login shells, and groups. |
