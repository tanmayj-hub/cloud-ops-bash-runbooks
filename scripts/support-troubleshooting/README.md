# Support Troubleshooting Scripts

Scripts for diagnostics collection, log review, connectivity checks, error summaries, and escalation support.

Scripts in this folder should use sanitized sample data and avoid collecting private information by default.

## Available Scripts

| Folder | Purpose |
| --- | --- |
| `port-connectivity-check/` | Check whether a host and port are reachable. |
| `log-error-summary/` | Summarize important error and warning lines from a log file. |
| `diagnostic-bundle-collector/` | Collect safe read-only diagnostics for support escalation. |
