# Docker Scripts

Scripts for Docker inspection, container reporting, and safe cleanup planning.

Cleanup-related scripts in this category are informational only unless explicitly reviewed for state-changing behavior.

## Available Scripts

| Folder | Purpose |
| --- | --- |
| `container-health-check/` | Show running containers and unhealthy containers. |
| `cleanup-dry-run/` | Show Docker resources that could be cleaned up without deleting anything. |
